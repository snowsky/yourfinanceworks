# Support Guide

We want to ensure you have the best experience with YourFinanceWORKS. Here's how you can get help.

## Documentation

Before reaching out, please check our comprehensive documentation:

- [Quick Start Guide](README.md#getting-started)
- [User Guide](docs/user-guide/GETTING_STARTED_GUIDE.md)
- [API Reference](docs/technical-notes/api_curl_examples.md) (Interactive docs available at `/docs` when running the API)

## Community Support

- **GitHub Issues**: For bug reports and feature requests, please use our [GitHub Issues](https://github.com/snowsky/yourfinanceworks/issues) page.
- **Discussions**: For general questions and "How-to" help, visit our [GitHub Discussions](https://github.com/snowsky/yourfinanceworks/discussions).

## Professional Support

For users with a **Commercial License**, you have access to priority support:

- **Email**: support@yourfinanceworks.com
- **SLA**: We aim to respond to all commercial inquiries within 24 business hours.

## Security Issues

If you discover a security vulnerability, please do **not** open a public issue. Instead, follow the instructions in our [SECURITY.md](SECURITY.md) file or email security@yourfinanceworks.com directly.
